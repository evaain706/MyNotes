
## 사용자 상호작용에 따른 스타일변경

- 마우스 hover, 클릭(active) 시의 효과
- input필드를 클릭하여 활성화(focus)시의 효과등

```jsx
function 상호작용에따른스타일변경() {
  return (
    <div className="space-y-4 p-4">
      {/* - 기본: bg-blue-500
        - hover: 마우스를 올리면 bg-blue-600
        - active: 클릭하는 순간 bg-blue-700, transform scale-95
      */}
      <button
        className="
          px-6 py-2 rounded-lg text-white font-semibold 
          bg-blue-500 
          transition-all duration-200 ease-in-out
          hover:bg-blue-600 
          active:bg-blue-700 active:scale-95
        "
      >
        호버 & 액티브상태
      </button>

      {/*
        - 기본: border-gray-300
        - focus: 클릭해서 입력 상태가 되면 border-violet-500, ring(그림자 효과) 추가
      */}
      <input
        type="text"
        placeholder="Focus on me"
        className="
          w-full px-4 py-2 rounded-lg 
          border-2 border-gray-300 
          transition-all duration-200
          focus:border-violet-500 focus:ring focus:ring-violet-200 focus:outline-none
        "
      />
    </div>
  );
}
```

## 컴포넌트의 상태에 따른 동적 스타일적용

- `state`의 값에 따라 `className`이 동적으로 부여되는 상황

```jsx
className={`
         rounded-lg text-white font-semibold transition-colors
          ${isSubmitting 
            ? 'bg-gray-400 cursor-not-allowed' // true일때는 커서이벤트 없애기
            : 'bg-green-500 hover:bg-green-600' // false일때는 hover시 변경
          }
        `}
```
위 코드와 같이 className내에서 템플릿리터럴을 사용하여 상태에따른 스타일 적용


```jsx

import { useState } from 'react';

function 상태에따른 스타일링() {
  const [isSubmitting, setIsSubmitting] = useState(false);
  const [selectedId, setSelectedId] = useState(2);
  const items = [{ id: 1, name: '하이1' }, { id: 2, name: '하이2' }];

  return (
    <div className="p-4">
      // isSubmitting state에 따른 동적 스타일링
      <button
        onClick={() => setIsSubmitting(true)}
        disabled={isSubmitting}
        className={`
         rounded-lg text-white font-semibold transition-colors
          ${isSubmitting 
            ? 'bg-gray-400 cursor-not-allowed' // true일때는 커서이벤트 없애기
            : 'bg-green-500 hover:bg-green-600' // false일때는 hover시 변경
          }
        `}
      >
        {isSubmitting ? '전송 중...' : '전송'}
      </button>

       // selectedId와 item.id가 일치하는 요소에만 선택적인 스타일 적용 
      <ul className="mt-4 space-y-2">
        {items.map((item) => (
          <li
            key={item.id}
            onClick={() => setSelectedId(item.id)}
            className={`
              p-3 rounded-lg cursor-pointer transition-all
              ${selectedId === item.id
                ? 'bg-blue-500 text-white shadow-md' // 선택된 경우
                : 'bg-gray-100 hover:bg-gray-200' // 선택되지 않은 경우
              }
            `}
          >
            {item.name}
          </li>
        ))}
      </ul>
    </div>
  );
}

```
