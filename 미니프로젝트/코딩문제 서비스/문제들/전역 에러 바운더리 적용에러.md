
## 문제


전역 에러바운더리를 적용했는데 렌더링에러시 FallBackUI가 아닌 화이트스크린이 보임


---


## 왜 화이트 스크린이었는지

throw Error (QuizScreen)

   ↓

React Router의 createBrowserRouter가 에러를 가로챔

   ↓

errorElement가 없었음 → 화이트 스크린

   ↓ (GlobalErrorBoundary까지 전파 안 됨)

## 수정 후

throw Error (QuizScreen)

   ↓

React Router가 에러를 가로챔

   ↓

errorElement: <GlobalErrorFallback /> → 에러 화면 표시 ✅
