
## 문제

###### 문제 설명

1부터 입력받은 숫자 n 사이에 있는 소수의 개수를 반환하는 함수, solution을 만들어 보세요.

소수는 1과 자기 자신으로만 나누어지는 수를 의미합니다.  
(1은 소수가 아닙니다.)

##### 제한 조건

- n은 2이상 1000000이하의 자연수입니다.

##### 입출력 예

|n|result|
|---|---|
|10|4|
|5|3|

## 첫풀이(시간초과)

```js
function solution(n) {
  let answer = 0;

  function getPrime(num) {
    if (num < 2) return false; 

    let count = 0;
    for (let i = 1; i <= num; i++) {
      if (num % i === 0) {
        count++;
      }
    }

    return count === 2; 
  }

  for (let i = 1; i <= n; i++) {
    if (getPrime(i)) {
      answer++;
    }
  }

  return answer;
}

```

O(n^2)의 시간복잡도가 나오는 코드인탓에 시간초과가 나왔다
