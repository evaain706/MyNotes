

### Array 메소드 정리

#### 1 배열 생성 

```js
let arr1 = [];                 // 빈 배열
let arr2 = [1, 2, 3];          // 숫자 배열
let arr3 = ["a", "b", "c"];    // 문자열 배열
let arr4 = [1, "a", true];     // 다양한 타입 혼합 가능

let arr5 = new Array();        // 빈 배열
let arr6 = new Array(5);       // 길이 5짜리 배열 (비어 있음, undefined 아님)
let arr7 = new Array(1, 2, 3); // [1, 2, 3]

//Array.of()
let arr8 = Array.of(5);        // [5]
//new Array(5)와 다르게 길이 5 배열이 아닌 [5] 생성.
let arr9 = Array.of(1, 2, 3);  // [1, 2, 3]



//Array.from()
let arr10 = Array.from("hello");       // ['h', 'e', 'l', 'l', 'o']
let arr11 = Array.from([1, 2, 3], x => x * 2); // [2, 4, 6]
let arr12 = Array.from({length: 5}, (_, i) => i + 1); // [1, 2, 3, 4, 5]
//유사 배열 객체나 이터러블을 배열로 변환할 때 유용.


//new Array().fill()
let arr13 = new Array(5).fill(0); // [0, 0, 0, 0, 0]
특정 값으로 초기화된 배열 생성 가능

// Array(n).keys()
let arr14 = [...Array(5).keys()]; // [0, 1, 2, 3, 4]
인덱스를 활용해 0부터 시작하는 배열 생성
```
